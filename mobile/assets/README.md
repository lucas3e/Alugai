# Assets do Alugai Mobile

Esta pasta contém os recursos visuais do aplicativo.

## Arquivos Necessários

Para que o aplicativo funcione corretamente, você precisa adicionar os seguintes arquivos nesta pasta:

### Ícones e Splash Screen

- **icon.png** (1024x1024px) - Ícone do aplicativo
- **adaptive-icon.png** (1024x1024px) - Ícone adaptativo para Android
- **splash.png** (1284x2778px) - Tela de splash
- **favicon.png** (48x48px) - Favicon para web

## Gerando os Assets

Você pode usar ferramentas online para gerar os assets:

1. **[App Icon Generator](https://www.appicon.co/)** - Gerar ícones
2. **[Figma](https://www.figma.com/)** - Criar designs personalizados
3. **[Canva](https://www.canva.com/)** - Criar splash screens

## Cores do Projeto

- **Primary**: #6200ee (Roxo)
- **Secondary**: #03dac6 (Ciano)
- **Background**: #f5f5f5 (Cinza claro)

## Dicas

- Use imagens PNG com fundo transparente para os ícones
- Mantenha o design simples e reconhecível
- Teste em diferentes tamanhos de tela
- Siga as diretrizes de design do Material Design

## Temporário

Enquanto você não tiver os assets personalizados, o Expo usará os ícones padrão.
